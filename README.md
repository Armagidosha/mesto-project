# Проект: Место

### Обзор
> Страничка, функционал которой пополнится в ближайшем будущем,  
где каждый сможет опубликовать свои творческие фото и лайкать фото других!👀
* JavaScript
* Grid Layout
* Flexbox
* Figma
* Сборка проекта с помощью WebPack
